Auto-Enaml Python¶

auto_enaml.__init__.auto_item(info, **kwargs)¶

Generate a Label-Editor combo tied to a particular Atom member.

Parameters:

info : tuple of (atom_instance, string) or a string

If used within an auto_view, a string can be used for the member name. Otherwise, a tuple with the atom_instance and the name must be used.

kwargs : dictionary

Keyword arguments to pass to the widgets. Label-specific properties can be set by using a prefix (e.g. label_tooltip). The same is true for the editor.
Returns:

item : AutoItem enaml instance
auto_enaml.__init__.auto_view(model, *objects, **kwargs)¶

Generate a view directly from an Atom instance.

Parameters:

model : Instance of Atom-based class.

objects : str or auto_items

Arguments used to build the auto_item editors. Can be a name of a member (string), or an auto_item.

kwargs : dictionary

Keyword arguments to pass to the view.
Returns:

item : AutoView enaml instance
auto_enaml.__init__.auto_window(model, *objects, **kwargs)¶

Generate a window directly from an Atom instance.

Parameters:

model : Instance of Atom-based class or an auto_view

objects : str or auto_items

Arguments used to build the auto_item editors. Can be a name of a member (string), or an auto_item.

kwargs : dictionary

Keyword arguments to pass to the window.
Returns:

item : AutoWindow enaml instance
